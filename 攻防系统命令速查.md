## 管理员命令 👑

### 系统管理
| 命令 | 说明 |
|------|------|
| `攻防 开` | 启用系统 |
| `攻防 关` | 禁用系统 |
| `攻防 查看` | 查看配置 |

### 参数配置
```
攻防 设置 参数 值
```

**可用参数：**
- `每日发起` - 最多发起次数（默认10）
- `每日拒绝` - 最多拒绝次数（默认10）
- `回合超时` - 超时时间(ms)（默认3600000）
- `默认回合` - 初始回合数（默认10）
- `逃脱率` - 逃脱成功率%（默认30）
- `伤害随机` - 随机范围（默认0=纯数值）
- `强制参战` - 是/否（默认否）
- `最小人数` - 最少参战人数（默认2）
- `手动开始` - 是/否（默认否）

**例：**
```
攻防 设置 逃脱率 50
攻防 设置 伤害随机 10-20
```

## 伤害计算公式

### 普通攻击
```
基础伤害 = ATK(攻击者)
防御减伤 = DEF(防守者) × 0.1
随机值 = [0~范围] (根据配置)
最终伤害 = max(1, 基础伤害 - 防御减伤 + 随机值)
```

### 伤害应用
```
护盾 -= min(护盾, 伤害)
如果护盾仍有余伤：HP -= (伤害 - 护盾减伤)
```

## 战斗生命周期

```
发起 → 待接受(Pending)
           ↓
        接受/拒绝
           ↓
   准备中(Preparing)
           ↓
    进行中(Ongoing)
           ↓
轮流操作：攻击/防守/逃脱
           ↓
  胜负决出(Ended)
           ↓
战斗结束，记录存档
```

## 常见操作流程

### 发起和加入战斗
```
玩家A: PK 发起 玩家B 玩家C
→ 战斗创建，ID: BATTLE_xxx

玩家B: PK 接受 BATTLE_xxx
玩家C: PK 接受 BATTLE_xxx
→ 战斗开始

当前轮到：玩家(敏捷最高)
```

### 进行战斗
```
当前玩家: 攻击 目标名字
→ 伤害计算，目标HP减少

下一玩家: 防守
→ 准备防守，等待攻击

后续玩家: 投降
→ 尝试逃脱（有成功率）
```

### 查看战况
```
任何玩家: 战斗状态
→ 显示所有参战者HP、护盾、存活状态

任何玩家: 战斗历史 BATTLE_xxx
→ 显示从头开始的所有行动
```
