# ⚔️ 攻防系统使用指南

## 系统概览
攻防（PK）系统是一个完整的回合制多人战斗系统，支持2+名玩家的自由混战。

## 玩家命令

### 1. 战斗管理
```
PK 发起                      # 发起开放战斗（其他人可加入）
PK 发起 玩家名 玩家名...     # 指定对手发起战斗（混战模式）
PK 接受 <战斗ID>             # 加入一场战斗
PK 拒绝 <战斗ID>             # 拒绝参加战斗
```

### 2. 战斗操作（轮到你时可用）
```
攻击 <对手名字>              # 发动普通攻击
防守                         # 进入防守姿态（增加防御）
投降                         # 尝试逃跑（有成功率限制）
逃跑                         # 同投降
```

### 3. 信息查询
```
属性                         # 查看自己的战斗属性
属性 <玩家名>               # 查看其他玩家的属性
战斗状态                     # 查看当前战斗情况
战斗状态 <战斗ID>           # 查看指定战斗情况
战斗历史 <战斗ID> [页码]    # 查看战斗行动记录
设置昵称 <昵称>             # 设置战斗中显示的昵称
```

## 管理员命令

### 1. 系统开关和配置
```
攻防 开                      # 启用攻防系统
攻防 关                      # 禁用攻防系统
攻防 查看                    # 查看当前所有配置
攻防 设置 参数 值            # 修改参数配置
```

#### 可配置参数：
- `每日发起` - 每位玩家每天最多可发起的战斗次数（默认10）
- `每日拒绝` - 每位玩家每天最多可拒绝的战斗次数（默认10）
- `回合超时` - 单个回合的超时时间（毫秒，默认3600000=1小时）
- `默认回合` - 新战斗的默认回合数（默认10）
- `逃脱率` - 投降逃脱的成功概率(0-100%)（默认30）
- `伤害随机` - 伤害值随机性（例："0-20"，默认0=纯数值）
- `强制参战` - 是否强制参加战斗（是/否）
- `最小人数` - 开始战斗的最少玩家数（默认2）
- `手动开始` - 是否需要管理员手动启动战斗（是/否）

### 2. 玩家管理
```
添加人员 <玩家名> [玩家2]... # 手动创建战斗并加入指定玩家
添加技能 <玩家名> <技能名>   # 为玩家解锁技能
属性 设置 <玩家> <属性> <值>  # 修改玩家的战斗属性
```

## 战斗属性系统

每个玩家拥有以下7个核心属性：

| 属性 | 英文 | 说明 | 默认值 |
|------|------|------|--------|
| 攻击力 | ATK | 每次攻击造成的基础伤害 | 50 |
| 防御力 | DEF | 减少受到的伤害（每点10%） | 30 |
| 敏捷 | AGI | 决定行动顺序 | 40 |
| 生命值 | HP | 玩家的健康值 | 100 |
| 临时护盾 | TMP_SHIELD | 优先受伤的盾值 | 0 |
| 魔法值 | MP | 使用技能的资源 | 50 |
| 恢复速率 | MP_REGEN | 每回合恢复的魔法值 | 5 |

## 战斗机制

### 初始化流程
1. 玩家发起战斗 → 战斗进入"待接受"状态
2. 其他玩家接受参加 → 战斗进入"准备中"状态
3. 达到最少人数或管理员启动 → 战斗进入"进行中"状态

### 回合顺序
- 按**敏捷(AGI)从高到低**排列行动顺序
- 每个回合轮流操作
- 击败的玩家自动跳过

### 伤害计算
```
基础伤害 = 攻击者ATK
防御减伤 = 防守者DEF * 0.1
最终伤害 = max(1, 基础伤害 - 防御减伤 + 随机值)
```

#### 伤害应用顺序
1. 优先扣除护盾(TMP_SHIELD)
2. 护盾不足则扣除生命值(HP)

### 逃脱机制
- 成功率可配置（默认30%）
- 成功逃脱：玩家被标记为"已败"，战斗继续进行
- 失败逃脱：回合直接结束，轮到下一个玩家

### 战斗结束条件
- 只有1名玩家存活 → **该玩家胜利** 🏆
- 同时出现多个平手情况 → **平手结束**

## 回合超时处理

设定的超时时间内，当前玩家未执行操作则：
- 系统自动跳过该玩家的回合
- 行动记录会标记为"超时"
- 战斗继续进行

## 日志和历史记录

### 完整记录内容
每一个战斗的所有行动都被完整记录，包括：
- 回合号
- 执行者和目标
- 行动类型（攻击/防守/逃脱）
- 伤害值和结果
- 时间戳

### 查看历史
```
战斗历史 <战斗ID>            # 第1页
战斗历史 <战斗ID> 2          # 第2页（每页10条记录）
```

## 常见问题

**Q: 怎样查看我的当前战斗状态？**
A: 输入 `战斗状态` 命令查看你参加的进行中战斗的详细信息。

**Q: 防守有什么具体效果？**
A: 防守会记录行动，在计算伤害时会应用防守者的DEF属性（每点减伤10%）。

**Q: 同时有多个战斗时怎么办？**
A: 系统会查找你参加的第一个进行中的战斗。建议在一个战斗结束后再参加其他战斗。

**Q: 超时会有惩罚吗？**
A: 仅会自动跳过该回合，没有额外惩罚。战斗继续进行。

**Q: 能否中途退出战斗？**
A: 可以通过 `投降` 命令尝试逃脱（需要成功率检定）。
