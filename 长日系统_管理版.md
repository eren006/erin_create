# ⚙️ 长日系统 - 管理员快速指南

## 系统概述

长日系统是一套完整的恋爱综艺游戏框架，包含 **18个功能模块、90+条命令**，支持角色管理、邀约系统、关系线、经济系统等核心功能。

---

## 📋 管理员权限等级

| 权限 | 功能 |
|------|------|
| **普通玩家** | 邀约、通讯、论坛、查询操作 |
| **普通管理员** | 创建角色、设置地点、发起官约、管理邀约等 |
| **超级管理员** | 管理管理员、重置数据、修改配置 |

## 🚀 初始化游戏（第一次搭建）

### 第1步：权限设置
```
授予管理员 角色名        # 为副管理员/统计员授予权限
清空管理员              # 【谨慎】重置所有管理员（需要超级权限）
管理员列表              # 查看当前管理员列表
```

### 第2步：创建角色
```
创建新角色 角色名        # 为每个玩家创建游戏角色
玩家名单               # 查看所有已创建的角色（玩家也可查看）
修改名字 旧名 新名       # 修改角色名
清除玩家 角色名         # 【谨慎】删除角色及其全部数据
```

### 第3步：配置地点
```
地点 查看              # 查看可用地点列表（玩家也可查看）
地点管理              # 管理地点权限
批量设置地点 地点 角色1 角色2...    # 批量分配地点权限
批量发放钥匙 地点 角色1 角色2...    # 批量发放私人房间钥匙
查看地点详情 地点名      # 查看地点具体信息和权限分配
```

### 第4步：群组和公告
```
添加群号 群号          # 添加关联的QQ群
查看群号              # 查看已配置的群号列表
群公告发布 公告内容     # 发布群公告
```

## 🎯 游戏进行中的关键管理任务

### 邀约管理（最重要）
```
发起官约 描述           # 发起官方约会（多人活动）
查看进行中             # 查看所有正在进行的邀约（需要定期检查！）
结束私约 邀约ID         # 结束超时的私约
提醒超时              # 提醒已超时的邀约
```

### 统计和数据收集
```
本场统计              # 查看当场邀约的互动统计
查看全员统计           # 查看所有角色的数据排行
存入统计              # 保存当前统计数据（备份）
```

### 剧情推进
```
统一送心动信 角色列表 内容    # 群发心动信（推进情节）
发起官约 描述           # 发起官方活动（如告白大会、特殊任务等）
设为npc 角色名          # 将某个角色设为NPC（由管理员控制）
```

### 时间管理
```
设置邀约时间 角色 时段1 时段2...   # 限制角色的邀约时段
清空邀约时间 角色      # 清除时间限制
查看锁定 / 时间锁定     # 查看当前时间限制配置
```

### 工具支持
```
随机分组 人数          # 随机分组（小游戏用）
同步名片              # 更新角色在系统中的信息
查看计时器            # 查看各项计时器运行状态
设置监听参数           # 配置系统监听和性能参数
```

## 💰 经济系统管理

### 礼物配置（玩家通过"礼物商城"购买，管理员配置清单）
```
上传预设礼物 礼物名*描述*价格     # 添加新礼物到商城（玩家可购买）
删除预设礼物 礼物ID    # 从商城移除礼物
```

**说明：** "礼物商城" 是玩家用来抽卡/购买的，管理员的职责是配置和维护礼物列表

### 拍卖系统（高价值物品交易）
```
添加拍卖物品 物品 初始价格        # 上架拍卖物品
删除拍卖物品 物品ID    # 取消拍卖
结算拍卖 物品ID       # 完成拍卖，给最高价出价者
```

## 📋 日常管理检查清单

### 每场邀约前
- [ ] 确认参与者名单
- [ ] 检查地点权限配置
- [ ] 设定邀约时间限制（如需要）

### 邀约进行中
- [ ] 定期查看"查看进行中"，监控邀约状态
- [ ] 关注"查看计时器"，防止超时

### 邀约结束后
- [ ] 执行"本场统计"收集数据
- [ ] 根据需要发送"统一送心动信"推进剧情
- [ ] 定期"存入统计"备份数据

### 定期维护
- [ ] 检查"管理员列表"确保权限正确
- [ ] 查看"论坛系统"了解舆论热点
- [ ] 监控"关系线统计"观察感情线进展
- [ ] 审核"群公告"是否需要更新

## ⚠️ 注意事项

### 危险操作（需谨慎）
- **清除玩家**：删除后所有数据无法恢复，包括邀约、关系线、信件等
- **清空管理员**：系统会失去所有管理员权限，需要配置文件恢复
- **清空关系线**：重置所有感情线数据

### 最佳实践
- 💾 **定期备份**：经常使用"存入统计"保存重要数据
- 🔍 **定期检查**：每场邀约都要看一次"查看进行中"
- 📊 **观察数据**：通过统计数据了解游戏运营状态
- 📝 **记录决策**：重要决定写下来方便后续复盘

## 🎯 典型工作流

### 启动新游戏版本
1. 创建所有角色 (`创建新角色`)
2. 配置地点权限 (`批量设置地点`)
3. 发放私人房间钥匙 (`批量发放钥匙`)
4. 发布游戏说明 (`群公告发布`)
5. 发起第一场官约 (`发起官约`)

### 常规运营
1. 监控邀约进行 (`查看进行中` - **每天检查**）
2. 处理超时邀约 (`提醒超时`)
3. 收集统计数据 (`本场统计`)
4. 发送官方心动信推进剧情 (`统一送心动信`)
5. 定期备份数据 (`存入统计`)

### 游戏收尾
1. 最终统计汇总 (`查看全员统计`)
2. 保存最终数据 (`存入统计`)
3. 清理过期数据 (如需要)

## 💡 管理技巧

### 如何维持游戏热度
- 定期发起官约，给玩家新的互动机会
- 通过心动信推进关键感情线
- 在论坛观察热点，了解玩家关注什么

### 如何处理邀约冲突
- 使用"设置邀约时间"限制时段，避免同时段有太多邀约
- 通过"结束私约"强制结束卡死的邀约
- 修改或删除出错的时间线

### 如何平衡经济
- 监控游戏币流向（统计数据）
- 定期调整礼物价格，维持消费平衡
- 利用拍卖系统回收玩家的游戏币
