# ⚔️ 装备系统使用指南

装备系统允许玩家装备武器、防具等来增强属性。

## 一、基础概念

### 装备槽位
```
头部（头盔、王冠等）
胸部（甲胄、衣服等）
手部（武器、盾牌等）
腿部（裤子、护腿等）
脚部（靴子、鞋子等）
```

### 装备属性加成
装备可以增加你的属性值。每件装备有两种属性加成方式：

**基础属性加成**：装备自带的固定属性提升
- 示例：铁制短剑 +15 攻击力

**强化属性加成**：通过强化等级增加的属性
- 示例：铁制短剑强化到+3级时，额外增加 +6 攻击力（3 × 2）
- 总加成 = 基础加成 + (强化等级 × 每级增量)

## 二、玩家命令

### 查看装备
```
查看装备
```
显示你当前穿着的所有装备及其属性加成。

**示例输出**：
```

🎩 头部: 铁制头盔 [+2] (DEF+12)
🛡️ 胸部: 皮革甲胄 [+5] (DEF+35,HP+100)
⚔️ 手部: 铁制短剑 [+3] (ATK+21)
👖 腿部: 空
👢 脚部: 敏捷靴子 [+1] (AGI+6)

📊 总属性加成:
· ATK+21
· DEF+47
· AGI+6
· HP+100
```

### 装备物品
```
装备 <装备名或代码>
```
穿上指定的装备。如果该槽位已有装备，旧装备会被卸下。

**示例**：
```
装备 EQUIP_001          # 按代码装备
装备 铁制短剑           # 按名称装备
```

**结果**：
```
✅ 你穿上了 铁制短剑！

属性加成: ATK+15

(原装备 皮革甲胄 已卸下)
```

### 脱装备
```
脱装备 <槽位>
```
卸下指定槽位的装备。

**槽位名**：head, chest, hand, leg, foot

**示例**：
```
脱装备 hand             # 卸下手部装备
脱装备 chest            # 卸下胸部装备
```

### 强化装备
```
强化装备 <槽位> [次数]
```
使用金币强化装备，提升其属性加成。

**参数**：
- 槽位：要强化的装备所在槽位
- 次数：可选，一次强化多少级（默认1）

**强化机制**：
- **成功率**：90%（可配置）
- **消耗**：100金币起，最高1000金币（按强化等级递增）
- **失败**：损失金币但装备保留
- **上限**：每件装备最多强化10级

**示例**：
```
强化装备 hand           # 强化手部装备1级，消耗100金币
强化装备 hand 5         # 强化手部装备5级，消耗共1000金币
```

**结果**：
```
⚡ 强化结果:

✅ 等级 0 → 1 强化成功！
✅ 等级 1 → 2 强化成功！
✅ 等级 2 → 3 强化成功！
❌ 等级 3 强化失败！消耗金币已扣除。
✅ 等级 4 → 5 强化成功！

💰 消耗金币: 1050
当前强化等级: 5/10
```

## 三、属性计算示例

### 示例 1：单装备加成
```
玩家基础属性：ATK 40
装备：铁制短剑 (ATK+15)
总属性：ATK 40 + 15 = ATK 55
```

### 示例 2：多装备加成
```
玩家基础属性：ATK 40, DEF 30, HP 80
装备 1：铁制短剑 (ATK+15)
装备 2：皮革甲胄 (DEF+20, HP+50)
装备 3：敏捷靴子 (AGI+5)

总属性：
- ATK: 40 + 15 = 55
- DEF: 30 + 20 = 50
- HP: 80 + 50 = 130
- AGI: 0 + 5 = 5
```

### 示例 3：强化加成
```
装备：铁制短剑
基础加成：ATK+15
强化等级：3
每级强化增量：ATK+2

总加成 = 15 + (3 × 2) = 21 ATK
```

## 四、装备获取方式

装备可以通过以下方式获得：

1. **商城购买** - 管理员上架的装备可直接购买
2. **池子抽取** - 从各类抽取池中随机获得
3. **合成制作** - 使用合成系统制作装备
4. **任务奖励** - 完成任务获得装备

## 五、常见问题

**Q: 装备后会自动增加属性吗？**
A: 是的，穿上装备后属性会立即增加，脱下后立即减少。

**Q: 能同时穿多件相同装备吗？**
A: 不能。每个槽位只能穿一件装备。但可以在不同槽位穿着相同类型的装备（如两件头部装备轮流穿）。

**Q: 强化失败会丢失装备吗？**
A: 不会。失败时只损失金币，装备保留。

**Q: 强化到10级后能继续强化吗？**
A: 不能。10级是最高等级。

**Q: 可以把强化等级转移到另一件装备上吗？**
A: 不能。强化等级绑定在具体的装备上。如果更换装备，新装备的强化等级是0。

**Q: 装备会损坏或掉耐久度吗？**
A: 不会。装备可以无限使用。

**Q: 强化成功率是固定的吗？**
A: 是的，默认90%。管理员可以修改这个概率。

## 六、装备和攻防系统

在攻防（PK）战斗中，你穿着的装备会自动应用其属性加成，提升你的战斗能力。

**示例**：
- 没装备：ATK 40, DEF 30
- 装备后：ATK 55 (铁剑+15), DEF 50 (皮甲+20)
- 在战斗中使用的属性就是装备后的值

## 七、最佳实践

1. **优先强化常用装备**
   - 如果一直用铁制短剑，就优先强化它
   
2. **平衡多个装备**
   - 不要只强化一件，多件装备均衡发展效果更好

3. **监控金币储备**
   - 强化是消耗金币的，确保有足够金币用于强化

4. **利用不同槽位**
   - 充分利用5个槽位，让每个部位都有装备

5. **等待掉落或获取更好装备**
   - 新装备可能更强，不要无谓强化劣质装备
