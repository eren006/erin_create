# 📮 长日写信综 - 完整使用指南

一个独立的正式信件系统，支持格式化书信、写信币赏金和配置管理。

---

## 系统概览

### 核心功能

| 功能 | 命令 | 说明 |
|------|------|------|
| 启用系统 | `启用写信综` | 【管理员】启用/禁用系统，自动注册写信币和特殊道具 |
| 发送信件 | `发送信件` | 发送正式、格式化的书信 |
| 施加道具 | `施加` | 对目标施加望远镜或羽毛笔效果 |
| 审核修改 | `羽毛笔修改` | 修改被拦截的信件内容后发送 |
| 查看状态 | `信件状态` | 查看系统配置和个人额度 |
| 设置配置 | `信件设置` | 【管理员】配置系统参数 |

### 核心特性

📮 **发送信件**
- 正式、有格式的书信系统
- 支持日期、附件、自定义署名
- 可选写信币赏金机制
- 每日限额可配置
- 即时投递到收件人私人群

💰 **写信币**
- 自动注册的游戏货币
- 根据信件字数发放赏金
- 可在其他系统中使用（如商城、合成等）
- 可作为其他系统的消费货币

🔗 **系统联动**
- 启用写信系统后，电话功能自动禁用
- 发送心愿需要消费写信币（可选）
- 发送私约需要消费写信币（可选）

⚙️ **灵活配置**
- 管理员可自由调整所有参数
- 支持启用/禁用整个系统
- 赏金机制完全可选
- 心愿和私约的消费成本可独立配置

## 管理员命令

### 1. 启用写信综

#### 用法

```
。启用写信综 <开启/关闭>
```

#### 效果

**开启**：
- ✅ 启用发送信件功能
- ✅ 自动注册"写信币"货币
- ✅ 玩家可使用发送信件

**关闭**：
- ❌ 禁用发送信件功能
- ❌ 玩家无法使用发送信件

#### 示例

```
。启用写信综 开启
✅ 写信综已启用！

✨ 已自动注册货币：写信币
📮 玩家可以开始使用「发送信件」命令。

。启用写信综 关闭
❌ 写信综已禁用。

玩家无法使用「发送信件」命令。
```

### 2. 信件设置

#### 用法

```
。信件设置 <参数> <值>
```

#### 参数说明

| 参数 | 说明 | 默认值 | 范围 |
|------|------|--------|------|
| 日限 | 每日最多可发送的信件数 | 5 | 1-999 |
| 赏金 | 每封信获得的写信币数（0为禁用） | 0 | 0-999 |
| 最小字数 | 获得赏金的最少字数（0为无限制） | 0 | 0-999 |
| 心愿成本 | 发送心愿需要消费的写信币数（0为禁用） | 0 | 0-999 |
| 私约成本 | 发送私约需要消费的写信币数（0为禁用） | 0 | 0-999 |

#### 示例

```
。信件设置 日限 10
✅ 已设置 日限 = 10

。信件设置 赏金 5
✅ 已设置 赏金 = 5

。信件设置 最小字数 10
✅ 已设置 最小字数 = 10

。信件设置 心愿成本 3
✅ 已设置 心愿成本 = 3

。信件设置 私约成本 5
✅ 已设置 私约成本 = 5
```

## 玩家命令

### 0. 施加特殊道具

#### 基础用法

```
。施加 <目标角色> <道具名>
```

#### 道具说明

| 道具 | 效果 | 说明 |
|------|------|------|
| 望远镜 | 自动抄录 | 当目标发出信件时，自动抄录一份给你 |
| 羽毛笔 | 拦截审核 | 当目标发出信件时，先拦截给你审核修改后再发出 |

#### 望远镜示例

```
。施加 小明 望远镜
✅ 你已向「小明」施加了「望远镜」！
```

**效果触发**：
- 小明发信给张三 → 你自动收到抄录一份
- 小明发信给你 → 收到提醒："📺 望远镜提醒：小明 正好发给了你（所以没有进行抄录）"

#### 羽毛笔示例

```
。施加 小红 羽毛笔
✅ 你已向「小红」施加了「羽毛笔」！
```

**效果触发**：
- 小红尝试发信给李四 → **拦截**，进入待审状态
- 小红尝试发信给你 → 正常发送，不进行拦截

#### 特殊情况处理

##### 望远镜：收件人是施加人

```
[施加人a向b施加了望远镜]

b发送信件给a：
✉️ 信件已送达「a」！
📺 望远镜提醒：正好发给了你（所以没有进行抄录）
```

##### 羽毛笔：收件人是施加人

```
[施加人a向b施加了羽毛笔]

b发送信件给a：
✉️ 信件已送达「a」！
[信件正常发送，不进行拦截]
```

### 1. 发送信件（正式书信）

#### 基础用法

```
。发送信件
【收件人】小明
【内容】亲爱的小明，今天天气真好...
```

#### 完整格式

```
。发送信件
【收件人】角色名
【内容】信件内容
【日期】日期（选填）
【附件】附加内容（选填）
【署名】落款（选填，默认为你的角色名）
```

#### 参数说明

| 参数 | 是否必填 | 说明 | 示例 |
|------|---------|------|------|
| 收件人 | ✅ | 接收信件的角色名 | 小明 |
| 内容 | ✅ | 信件正文 | 你好啊，好久不见... |
| 日期 | ❌ | 显示在信件顶部 | 2026年4月28日 |
| 附件 | ❌ | 额外备注，以分隔线显示 | 随信附赠礼物一份... |
| 署名 | ❌ | 落款，默认为你的角色名 | 小红敬上 |

#### 示例 1：简单信件

```
。发送信件
【收件人】小明
【内容】嗨！你好吗？
```

✅ 回复：
```
✉️ 信件已送达「小明」！
🖋️ 落款：你的角色名
📅 D123（今日剩余：4/5）
```

#### 示例 2：完整信件

```
。发送信件
【收件人】小明
【内容】亲爱的小明，

你忠诚的朋友

【署名】小红敬上
```

✅ 收件人的私人群会收到：
```
✉️ 小明，你收到一封信：

📅 日期：2026年4月28日

「亲爱的小明，

我们已经很久没有见面了。今天我想起了我们过去一起玩耍的日子，时光真的过得很快。希望你最近一切安好，期待我们能很快再次见面。

你忠诚的朋友」

附件：
--------------------
随信附赠: 一张泛黄的照片，记录了我们美好的回忆
```

#### 赏金机制

如果管理员启用了写信币赏金：
- 需要达到**最少字数**才能获得赏金
- 每封信获得**固定写信币**数量
- 写信币可用于其他系统中

**示例（启用赏金后）**：
```
✉️ 信件已送达「小明」！
🖋️ 落款：小红敬上
📅 D123（今日剩余：4/5）
💰 写信币 +5（共 50）
```

**字数不足**：
```
✉️ 信件已送达「小明」！
🖋️ 落款：小红敬上
📅 D123（今日剩余：4/5）
📝 提示：字数不足 20，未获得赏金。
```

#### 投递规则

- 信件**立即发送**到收件人的私人群
- 收件人会收到 **@** 提示
- 如果启用赏金机制，**立即发放**写信币
- 超过日限额后无法发送

#### 错误提示

| 提示 | 原因 | 解决方案 |
|------|------|---------|
| 发送信件功能未启用 | 管理员未启用 | 等待管理员执行「启用写信综 开启」 |
| 请先使用「创建新角色」 | 没有绑定角色 | 执行「创建新角色」创建角色 |
| 请指定【收件人】 | 没有填写收件人 | 按格式填写【收件人】字段 |
| 找不到角色「XXX」 | 收件人名字错误或不存在 | 检查角色名是否拼写正确 |
| 信件内容不能为空 | 没有填写内容 | 按格式填写【内容】字段 |
| 今日已发 5/5 封信件 | 超过日限额 | 等待游戏日重置 |

#### 特殊道具触发

如果你发送信件时被对方施加了特殊道具，系统会自动处理：

**被施加望远镜**：
```
⏳ 你的信件正在被对方通过望远镜监控...
```

**被施加羽毛笔**：

### 2. 羽毛笔审核修改

#### 用法

```
。羽毛笔修改 <修改后的内容>
```

#### 说明

当有角色对你施加了羽毛笔效果时，你会收到待审通知：

```
🖋️ 【羽毛笔待审】小明 想向 张三 发送信件：
「原信内容...」

请使用 。羽毛笔修改 <新内容> 修改内容后发送。
⏰ 3小时内未修改将自动发送原文。
```

你可以修改内容后发送：
```
。羽毛笔修改 这是修改后的内容

✅ 信件已发送！已以「小明」的名义发给「张三」。
```

#### 超时处理

- 如果3小时内没有修改，系统会**自动发送原文**给原来的收件人
- 待审信件会自动清除
- 信件末尾会标记：`(此信件已超时3小时，自动发送原文)`

#### 优先级说明

如果同时被多个人施加了望远镜和羽毛笔：
- **按施加时间的先后顺序执行**
- 先施加的效果优先处理
- 后续效果会排队等待处理

## 常见问题

### 系统启用

**Q: 如何启用写信综？**
A: 管理员执行「启用写信综 开启」即可。系统会自动注册"写信币"货币。

**Q: 启用后玩家立即可用吗？**
A: 是的，启用后玩家立即可以使用「发送信件」命令。

**Q: 如何禁用写信综？**
A: 管理员执行「启用写信综 关闭」。玩家会无法使用发送信件。

### 关于发送信件

**Q: 信件会立即送达吗？**
A: 是的，信件会立即发送到收件人的私人群，并会@收件人。

**Q: 能否修改已发送的信件？**
A: 不能。如果需要修改，重新发送一封新信件。

**Q: 赏金是自动发放的吗？**
A: 是的，如果达到最少字数要求，赏金会自动发放并添加到写信币属性中。

**Q: 写信币有什么用？**
A: 这是一个标准的游戏属性/货币，可以在其他系统中使用（如商城、合成、任务奖励等）。

### 关于日限额

**Q: 日限额在什么时候重置？**
A: 每当游戏日期变化时（D123 → D124）自动重置。

**Q: 如何查看还能发多少封？**
A: 执行「信件状态」查看今日已用和剩余额度。

**Q: 超过限额后怎么办？**
A: 只能等待游戏日重置，无法立即发送。

### 关于赏金

**Q: 最小字数是指什么？**
A: 信件正文（【内容】字段）的字数，不包括日期、附件、署名。

**Q: 空格和标点算字数吗？**
A: 不算。只计算实际的文字字数。

**Q: 赏金是一次性的吗？**
A: 是的，每封信只能获得一次赏金。

### 关于系统联动

**Q: 启用写信系统后会怎样？**
A: 系统会自动注册"写信币"货币。同时会发生以下变化：
- 电话功能自动禁用，玩家无法使用「电话」命令
- 如果配置了心愿成本和私约成本，发送心愿和私约时需要消费写信币

**Q: 为什么启用写信系统后电话功能被禁用了？**
A: 这是系统的设计机制。启用写信系统表示启用了新的沟通方式，旧的电话功能会被替代。

**Q: 心愿和私约需要消费写信币吗？**
A: 取决于管理员的配置。如果设置了「心愿成本」或「私约成本」大于0，则需要消费。如果成本为0（默认），则不消费。

**Q: 发送心愿/私约时写信币不足怎么办？**
A: 玩家需要先获得足够的写信币。可以通过发送信件（如果启用了赏金）来获得写信币。系统会显示需要多少枚以及当前余额。

**Q: 能否只禁用电话功能而不禁用写信系统？**
A: 不能。启用写信系统就会同时禁用电话功能，这是一个整体的系统联动。如果不想禁用电话，就不要启用写信系统。

**Q: 心愿和私约的消费成本怎么设置？**
A: 使用「信件设置」命令设置：
```
。信件设置 心愿成本 3
。信件设置 私约成本 5
```
设置为0表示禁用消费。

### 关于特殊道具

**Q: 望远镜和羽毛笔是什么？**
A: 这是两个特殊的写信道具：
- **望远镜**：施加后能抄录对方发出的信件
- **羽毛笔**：施加后能拦截对方发出的信件进行审核修改

**Q: 如何获得望远镜和羽毛笔？**
A: 这两个道具可以通过以下方式获得：
- 从配置的物品池子中随机掉落
- 从商城中购买（需要管理员上架）
- 其他系统奖励（如任务、活动等）

**Q: 这两个道具只有启用写信系统后才能上架吗？**
A: 是的。望远镜(SPEC_003)和羽毛笔(SPEC_004)只有在启用写信系统后，才能：
- 添加到池子中
- 上架到商城中

如果尝试在未启用写信系统时操作这两个道具，系统会拒绝并提示。

**Q: 望远镜会被对方发现吗？**
A: 不会被发现。望远镜抄录是自动进行的，对方看不出来自己的信件被抄录了。

**Q: 被施加了望远镜，收件人是施加人的情况是什么意思？**
A: 举个例子：
- a对b施加了望远镜
- b发送信件给a本人
- 这时不需要抄录（因为a已经直接收到了信件）
- 但系统会显示提醒："📺 望远镜提醒：b 正好发给了你"
- 这样让玩家知道望远镜功能被触发了，只是因为特殊情况没有额外效果

**Q: 羽毛笔拦截后，原发件人知道吗？**
A: 发件人会看到信件已被拦截的提示：
```
⏳ 你的信件已被对方拦截，等待修改或超时发送...
```

**Q: 如果羽毛笔拦截的信件被修改了，收件人能看出来吗？**
A: 不能。收件人只会看到修改后的内容，看不出来是否被修改过。除非在信件中明确提到，否则收件人不会知道。

**Q: 3小时自动发送的是原文吗？**
A: 是的。如果施加人(拦截者)在3小时内没有修改，系统会自动发送**原始内容**给原来的收件人。

**Q: 同时被施加了望远镜和羽毛笔怎么办？**
A: 会按照**施加的时间顺序**执行：
- 先施加的效果先处理
- 后施加的效果排队等待

例如：
- 10:00 施加望远镜
- 10:30 施加羽毛笔
- 发信时：先处理望远镜(抄录)，再处理羽毛笔(拦截)

**Q: 能否对同一个人多次施加同一个道具？**
A: 可以。同一个人可以被多个人分别施加望远镜或羽毛笔。

例如：
- a对b施加望远镜
- c也对b施加望远镜
- b发信时，a和c都会收到抄录

**Q: 道具施加后会消失吗？**
A: 是的。一旦施加成功，使用的道具就会直接消失。

**Q: 如果没有合适的道具，施加会失败吗？**
A: 是的。施加时系统会检查你是否拥有该道具。如果没有或数量为0，施加会失败并提示：
```
❌ 你没有「望远镜」。
```

## 最佳实践

### 对于管理员

1. **初期配置**
   ```
   。启用写信综 开启
   。信件设置 日限 5
   。信件设置 赏金 5
   ```
   这是一个安全、平衡的起点。

2. **启用系统联动**（可选）
   如果想让心愿和私约需要成本，添加：
   ```
   。信件设置 心愿成本 3
   。信件设置 私约成本 5
   ```
   **注意**：启用后，电话功能会被禁用。

3. **根据反馈调整**
   - 如果玩家觉得赏金太少，增加赏金数量
   - 如果担心经济过剩，提高最小字数要求
   - 如果日限额过高，适度降低
   - 如果心愿/私约成本过高导致玩家无法参与，降低成本或增加赏金

4. **与其他系统集成**
   - 写信币可用于商城购物
   - 写信币可作为合成材料
   - 写信币可用于任务奖励
   - 写信币可用于心愿/私约的启动成本（系统联动）

### 对于玩家

1. **充分利用格式**
   - 使用日期让信件更有故事感
   - 使用附件补充重要信息
   - 自定义署名体现个性

2. **有目的地写信**
   - 为获得赏金而精心构思（如果启用赏金）
   - 利用信件加强人物互动
   - 创造游戏内的故事线索

3. **检查日限额**
   - 定期执行「信件状态」了解额度
   - 合理分配每日的信件配额
   - 重要内容优先发送

4. **谨慎使用特殊道具**
   - **望远镜**：用于监控特定角色，但要注意不要滥用破坏游戏体验
   - **羽毛笔**：用于管制或检查他人信件内容，使用前应考虑人物关系的合理性
   - 两个道具都很消耗资源，要权衡使用价值
