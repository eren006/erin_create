# 🎮 长日系统 - 玩家快速指南

## 欢迎来到长日！

这是一个恋爱综艺类游戏系统，你将创建一个角色，通过邀约、互动、建立感情线来体验虚拟综艺生活。

---

## 📋 核心概念速览

| 概念 | 说明 |
|------|------|
| **邀约** | 与他人约定时间和地点进行互动 |
| **关系线** | 记录你与他人之间的感情关系 |
| **心动信** | 表达感情的特殊信件 |
| **论坛** | 讨论和评论游戏内的故事 |
| **礼物** | 购买和赠送，表达心意 |
| **时间线** | 你已确认的所有邀约日程 |

## 💡 新手必知

### 邀约类型
- **私约**：两人私密互动（只有双方知道）
- **官约**：官方活动（多人参加，公开进行）
- **心愿**：开放式邀请（你发布后，他人主动接受）
- **多人邀约申请**：加入他人的多人邀约（申请→对方同意→正式加入）

### 感情线建立
1. 你对他人感兴趣 → 拉线（A→B）
2. 他也对你感兴趣 → 他拉线（B→A）
3. 你们互相确认 → 建立双向关系线 ❤️

### 关键提示
- 💌 **心动信很重要**：发送心动信会被记录，影响你的人气和感情线发展
- 🎁 **礼物要花钱**：购买礼物消耗游戏币，邀约可以获得游戏币
- 📊 **论坛是舆论场**：论坛讨论影响谁是"人气角色"和热门CP
- ⏰ **时间线很重要**：邀约确认后会写入你的时间线，形成互动历史

### 获得游戏币的方式
- 参加邀约获得奖励
- 完成特定任务
- 论坛点赞多的帖子可能有奖励

## 🚀 快速上手流程

1. **创建角色** → 你的游戏身份已准备好
2. **查看地点和玩家** → 了解地点和可互动的角色
3. **发起邀约/心愿或申请加入** → 开始与他人互动
4. **建立关系线** → 表达你的感情和拉线
5. **发送心动信** → 用特殊方式表达情感
6. **参与论坛讨论** → 评价和讨论游戏内的故事
7. **收集礼物** → 购买和赠送，增加互动仪式感

## ❓ 常见问题

**Q: 私约、心愿、申请加入有什么区别？**
A: 
- 私约：你直接邀请特定人物，两人私密互动
- 心愿：你发布开放邀请，任何人都可以接受
- 申请加入：你加入他人已发起的多人邀约（需要对方同意）

**Q: 怎样和某人建立正式感情线？**
A: 你拉线给他，他也拉线给你，然后双方都确认，就形成了双向感情线。

**Q: 别人申请加入我的邀约，我怎么处理？**
A: 用"加入请求"查看所有申请，然后用"同意加入"或"拒绝加入"来处理。

**Q: 我的时间线有什么用？**
A: 时间线记录了你所有已确认的邀约日程，方便你查看和管理。

**Q: 如何成为热门角色？**
A: 积极参与邀约、发送心动信、发表论坛评论，关系线多、被讨论多的角色自然人气高。

**Q: 可以取消已发送的心动信吗？**
A: 可以用"撤回心动信"命令，但如果已被看过可能为时已晚。
