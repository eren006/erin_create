## 💡 优化建议按指令分类

### 高度重复的指令模式
1. **查询类** (20+个) - 都是读取数据 + 格式化
   - 可统一为：`showData(category, filters)`

2. **设置类** (15+个) - 都是验证 + 保存 + 反馈
   - 可统一为：`updateSetting(key, value, validator)`

3. **删除类** (8个) - 都是权限检查 + 删除 + 反馈
   - 可统一为：`deleteItem(type, id, validator)`

4. **权限管理** (8个) - 都是权限检查 + 修改管理员 + 反馈
   - 可统一为：`managePermission(action, target)`

5. **时间管理** (6个) - 都是时间验证 + 日期操作
   - 可统一为：`manageTime(action, day, time, duration)`

### 可以合并的指令
1. "看心愿" + "我的图鉴" → 统一为 "心愿" 指令
2. "删除时间线" + "废除时间线" + "修改时间线" → 统一为 "时间管理"
3. "查看地点详情" + "查看钥匙分配" → 统一为 "地点信息"
4. "添加拍卖物品" + "删除拍卖物品" → 统一为 "拍卖物品"
