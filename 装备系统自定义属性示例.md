# ⚔️ 装备系统自定义属性使用示例

装备系统支持使用任何已注册的属性，不限于HP、MP、ATK、DEF、AGI这些基础属性。

## 前置要求

**所有属性必须提前注册**，才能在装备中使用。

使用命令注册属性：
```
我创建属性 <属性名> [最小值] [最大值] [默认值]
```

## 快速开始

### 步骤 1: 创建自定义属性

```
我创建属性 智力 0 100 30
我创建属性 体质 0 100 40
我创建属性 精神 0 100 35
我创建属性 运气 0 100 25
我创建属性 魅力 -50 50 0
我创建属性 暴击率 0 100 10
我创建属性 抗性 0 100 0
```

### 步骤 2: 使用这些属性创建装备

```
注册装备 智者法杖*法术强化*hand*智力+20,MP+50*智力+3*10
注册装备 体质铠甲*防御装甲*chest*体质+30,HP+100*体质+5*10
注册装备 精神头盔*精神加强*head*精神+25,MP+30*精神+2*10
注册装备 幸运戒指*增加运气*ring*运气+15*运气+2*8
注册装备 魅力披风*贵族披风*chest*魅力+20,ATK+10*魅力+1*10
```

## 最佳实践

### 1. 属性设计
- **基础属性**：HP、MP、ATK、DEF、AGI（必有）
- **职业属性**：法术强度、格挡率、治疗力等（职业特定）
- **特殊属性**：暴击、闪避、冷却缩减等（增强多样性）

### 2. 平衡设计
```
# 不推荐：一个装备加太多属性
注册装备 超级武器*过强*hand*ATK+100,DEF+100,MP+100,暴击+100

# 推荐：专精设计
注册装备 暴击剑*专精暴击*hand*ATK+40,暴击+80*ATK+3*10
注册装备 防守盾*专精防守*hand*DEF+50,格挡+60*DEF+5*10
```

### 3. 强化策略
```
# 基础属性和强化属性可以不同
注册装备 法师杖*法术武器*hand*ATK+15,法术+50*法术+8*10
# 这样强化时ATK加少，法术加多，符合法师特性
```

## 常见问题

**Q: 能用没注册过的属性吗？**
A: 不能。注册装备时会验证所有属性是否已注册，如果有未注册属性会出现错误。

**Q: 能修改已注册属性的范围吗？**
A: 不能直接修改。如果需要改变属性范围，建议用新属性替代。

**Q: 一件装备最多能有多少个属性？**
A: 没有限制，理论上可以无限多个。但为了平衡，建议1-4个属性。

**Q: 强化属性必须和基础属性相同吗？**
A: 不必须。可以基础加ATK，强化加暴击，这样可以设计更灵活的装备。

**Q: 属性值可以是负数吗？**
A: 可以。例如 `诅咒物品*坏装备*hand*ATK-20,DEF-10` 会降低属性（虽然不推荐）。

## 与攻防系统的兼容

自定义属性在穿戴装备时会自动应用到玩家属性，包括在攻防战斗中使用。

例如，如果创建了这样的装备：
```
注册装备 战神铠*强化装备*chest*战斗力+50,ATK+30,DEF+20
```

穿上后，你的战斗力、ATK、DEF都会增加，在PK战斗中也会使用增强后的属性值。
